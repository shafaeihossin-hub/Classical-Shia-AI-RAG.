## ðŸ”¬ Project Overview
This research introduces a localized RAG system specialized for theological discourse. Using an Agentic Query Expansion mechanism with DeepSeek-R1, the system is capable of high-precision cross-referencing across the Holy Quran, Al-Kafi, Nahj al-Balagha, and Musnad Ahmad ibn Hanbal.

### Key Features:
- Privacy-First: Localized architecture ensures data privacy via Docker and Qdrant.
- Agentic Search: Improves retrieval hit rates by generating context-aware search keywords before querying the vector database.
- Anti-Hallucination: Strictly adheres to verified local contexts to prevent common LLM hallucinations.
