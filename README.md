## ðŸ”¬ Project Overview
This research introduces a localized RAG system specialized for theological discourse. [span_3](start_span)Using an Agentic Query Expansion mechanism with DeepSeek-R1, the system is capable of high-precision cross-referencing across the Holy Quran, Al-Kafi, Nahj al-Balagha, and Musnad Ahmad ibn Hanbal[span_3](end_span).

### Key Features:
- [span_4](start_span)Privacy-First: Localized architecture ensures data privacy via Docker and Qdrant[span_4](end_span).
- [span_5](start_span)Agentic Search: Improves retrieval hit rates by generating context-aware search keywords before querying the vector database[span_5](end_span).
- [span_6](start_span)Anti-Hallucination: Strictly adheres to verified local contexts to prevent common LLM hallucinations[span_6](end_span).
